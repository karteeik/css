COMPUTER

A computer is a machine that can store, process, and output data, and perform logical and arithmetic operations automatically. Computers come in many different sizes and shapes, from handheld smartphones to supercomputers that weigh more than 300 tons. Computers use a binary system, which uses the numbers 0 and 1 to complete tasks like storing data, calculating algorithms, and displaying information.
